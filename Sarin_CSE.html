Scratch Project

Aug 30, 2017 - Sept 7, 2017

Description: The main objective of this project was to demonstrate our knowledge on how to use scratch by using it to create a game. Prior to and during the project, I learned how to create games using scratch. My specific role in the project was to help with designing the game and coding it on scratch. I also helped with the documentation as well. Our game was a simple jumping game with shapes where the user's objective is to jump over obstacles. It is very similar to the Google dinosaur game. The biggest challenge was that we were having trouble figuring out a way to make the obstacle repeatedly show up in a loop until the user touches the obstacle. We only used one obstacle sprite because it was much easier then making many multiple copies of the same sprite and adding it to our code. We overcame the problem effectively and found a way for the obstacle to regenerate after it left the screen to give it the illusion that there were multiple obstacles in a row.
